# Cashflow Compass

Стартовий каркас для вебдодатка моніторингу витрат з фокусом на:

- адаптивний UI для мобільних екранів;
- App Router на Next.js;
- готовність до PWA/Android інтеграції;
- подальше підключення бази даних, авторизації та аналітики.

## Стек

- Next.js 15
- React 19
- TypeScript

## Що вже є

- стартова landing/dashboard сторінка;
- візуальна структура для витрат, бюджетів та короткої аналітики;
- `manifest.webmanifest` і іконки для PWA;
- базова конфігурація проєкту.

## Як запустити

Після встановлення Node.js:

```bash
npm install
npm run dev
```

Потім відкрий `http://localhost:3000`.

## Наступні кроки для MVP

1. Додати форму створення транзакцій.
2. Підключити PostgreSQL + Prisma.
3. Додати авторизацію.
4. Зробити збереження бюджетів і аналітики.
5. Підключити service worker для повноцінного PWA режиму.
